# ✅ Портфолио – Полный список реализованного

## 🎯 Задача

Создать страницу **"Мое портфолио"** в Next.js (App Router) с использованием Sanity CMS, где администратор может полностью управлять контентом без изменения кода.

---

## 🔍 Что было реализовано

### 1. ✅ Backend (Sanity CMS)

#### Новая схема `Portfolio`

- **Файл**: `sanity/schemaTypes/portfolio.js` ✅
- **Поля**:
  - `title` (localeString) - многоязычный заголовок ✅
  - `subtitle` (localeString) - многоязычный подзаголовок ✅
  - `documents` (array) - массив документов ✅
    - `title` (localeString) - название документа ✅
    - `description` (localeText) - описание документа ✅
    - `file` (file) - загрузка PDF/DOC/DOCX ✅
- **Валидация**: Обязательные поля отмечены ✅
- **Preview**: Встроенный preview в Sanity Studio ✅

#### Регистрация схемы

- **Файл**: `sanity/schemaTypes/index.js` ✅
- **Изменение**: Добавлен импорт и экспорт `portfolio` ✅

#### GROQ Query и функции

- **Файл**: `sanity/lib/queries.js` ✅
- **Добавлено**:
  - `PORTFOLIO_QUERY` - GROQ запрос ✅
  - `getPortfolioData()` - fetch функция ✅
  - `mapPortfolioData()` - трансформация данных ✅
- **Логика**:
  - Получает данные из Sanity ✅
  - Определяет тип файла (PDF/Word) ✅
  - Локализует контент для каждого языка ✅
  - Возвращает структурированные данные ✅
