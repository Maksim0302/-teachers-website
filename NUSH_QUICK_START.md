# 📚 НУШ (Нова українська школа) – Quick Start

> **Хотите быстро начать работу с НУШ документами? Это руководство для вас!**

---

## 🚀 3 простых шага

### Шаг 1️⃣: Создать НУШ документы в Sanity

1. Откройте **Sanity Studio**
2. Нажмите **"+ Create"** (или найдите **NUŠ** в левом меню)
3. Создайте новую запись

### Шаг 2️⃣: Заполнить информацию

```
Title (Название):
  EN: "NUŠ Documents"
  UK: "НУШ Документи"
  RU: "НУШ Документы"

Subtitle (Подзаголовок) - опционально:
  EN: "Nova Ukrainska Shkola resources"
  UK: "Матеріали Нової Української Школи"
  RU: "Материалы Новой Украинской Школы"
```

### Шаг 3️⃣: Добавить документы

1. Нажмите **"+ Add item"** в **"NUŠ Documents"**
2. Заполните для каждого документа:
   - **Document Title**: Название (например, "Методичні рекомендації")
   - **Document Description**: Описание (опционально)
   - **Document File**: PDF або Word файл (обязательно)
3. Нажмите **"Publish"**

## ❓ Часто задаваемые вопросы

**Q: Какие форматы поддерживаются?**  
A: PDF, DOC, DOCX

**Q: Сколько документов можно добавить?**  
A: Неограниченно! Добавляйте сколько нужно

**Q: Как удалить документ?**  
A: Удалите элемент из массива и нажмите "Publish"

**Q: Можно ли предпросмотреть PDF?**  
A: Да, для PDF файлов доступна кнопка "Preview" (откроется в новой вкладке)
