## 🎨 Компоненты UI

### 1️⃣ Карточка документа (hover state)

```
┌────────────────────────────────────┐
│ ┌──┐                               │
│ │📄│ Project Report 2024           │
│ └──┘ Comprehensive analysis...     │
│      PDF | report.pdf              │

│      PDF | report.pdf              │
│                     [Preview][Down]│ ← тени усилены
└────────────────────────────────────┘
```

---

### 2️⃣ Типы файлов

#### PDF документ

```
┌─────────────────────────────────────┐
│ 📄                                  │
│ Document Title                      │
│ Brief description...                │
│ PDF | filename.pdf                  │
│                                     │
│ [Preview]  [Download]               │
└─────────────────────────────────────┘
```

#### Word документ

```
┌─────────────────────────────────────┐
│ 📘                                  │
│ Document Title                      │
│ Brief description...                │
│ Word | filename.docx                │
│                                     │
│        [Download]                   │
└─────────────────────────────────────┘
```

**Разница:**

- PDF: есть кнопка Preview
- Word: нет кнопки Preview (Preview недоступен)

## ♿ Доступность

### Семантика

- ✅ `<section>` для основной области
- ✅ `<h1>` для заголовка
- ✅ `<h3>` для названия документа
- ✅ `<button>` для кнопок действий

### ARIA атрибуты

```html
<button aria-label="Preview PDF document">Preview</button>
```

### Клавиатурная навигация

- ✅ Tab - переход между кнопками
- ✅ Enter/Space - активация кнопки
- ✅ Все интерактивные элементы доступны

## ✅ Финальный результат

Пользователь видит:

1. **Заголовок и описание** портфолио
2. **Список документов** в виде красивых карточек
3. **Для каждого документа**:
   - Иконку типа файла
   - Название
   - Описание (если есть)
   - Тип и имя файла
   - Кнопки действий
4. **Возможность**:
   - Просмотра PDF
   - Скачивания файлов
   - Переключения языков
