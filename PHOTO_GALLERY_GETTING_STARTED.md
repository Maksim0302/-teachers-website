# 🚀 Фотогалерея - Инструкция по первому запуску

## Что создано

✅ Полнофункциональная страница **фотогалереи** с интеграцией Sanity CMS
✅ Адаптивная сетка фотографий (4-3-2-1 колонки)
✅ Lightbox модальное окно с навигацией
✅ Полная мультиязычность (Украинский, Русский, Английский)
✅ Оптимизированные изображения через Next.js
✅ Полная документация для администраторов и разработчиков

## 🔧 Первые шаги администратора

### 1. Откройте Sanity Studio

```
http://localhost:3000/studio
# или
npm run sanity
```

### 2. Создайте документ "Photo Gallery"

1. **В левом меню** найдите **"Photo Gallery"**
2. **Нажмите "+ Create"** (кнопка создания)
3. **Заполните поля**:

#### Заголовок (Title)

- **Украинский (uk)**: "Фотогалерея" (или ваше название)
- **Русский (ru)**: "Фотогалерея" (или ваше название)
- **Английский (en)**: "Photo Gallery" (или ваше название)

#### Подзаголовок (Subtitle) - опционально

- **Украинский**: "Подивіться нашу колекцію фотографій"
- **Русский**: "Посмотрите нашу коллекцию фотографий"
- **Английский**: "Browse our collection of photos"

### 3. Добавьте фотографии

1. **Прокрутите вниз** до раздела "Photos"
2. **Нажмите "+ Add item"**
3. **Для каждой фотографии**:
   - 📸 **Upload image** - выберите файл с компьютера
   - 📝 **Alt** (обязательно) - описание фото (например: "Учащиеся на уроке")
   - 💬 **Caption** (опционально) - подпись под фото

### 4. Опубликуйте

1. **Нажмите "Publish"** (кнопка в верхнем правом углу)
2. **Готово!** Фотографии появятся на сайте

## 🌐 Где смотреть результат

После публикации откройте в браузере:

```
https://your-domain.com/photo-gallery
https://your-domain.com/uk/photo-gallery (Украинский)
https://your-domain.com/ru/photo-gallery (Русский)
https://your-domain.com/en/photo-gallery (Английский)
```

## 📱 Как это выглядит

### На компьютере

- 4 фотографии в ряд
- Плавные эффекты при наведении (лупа)

### На планшете

- 3 фотографии в ряд

### На телефоне

- 2 фотографии в ряд (большие экраны)
- 1 фотография в ряд (маленькие экраны)

### При клике на фотографию

- Открывается **полноэкранное модальное окно**
- Видны **стрелки для навигации** ← →
- Показывается **счетчик фотографий** (1/10)
- Видна **подпись под фотографией** (если заполнена)
- **Закрытие**: нажать Esc, ✕, или стрелки

## ⌨️ Клавиатурная навигация в Lightbox

| Клавиша         | Действие               |
| --------------- | ---------------------- |
| **Esc**         | Закрыть модальное окно |
| **→**           | Следующая фотография   |
| **←**           | Предыдущая фотография  |
| **Клик на фон** | Закрыть окно           |

## 📝 Рекомендации при загрузке фотографий

### Размер изображений

- Минимум: 1000×750 px
- Оптимально: 1920×1440 px
- Формат: JPG (меньше размер), PNG, WebP

### Альтернативный текст (Alt)

Правильный:

- ✅ "Учащиеся работают в группах"
- ✅ "Класс на уроке математики"
- ✅ "Школьное мероприятие на открытом воздухе"

Неправильный:

- ❌ "Image 1"
- ❌ "Photo"
- ❌ Спам с ключевыми словами

### Подписи (Captions)

Отличные подписи:

- "Уроки математики, сентябрь 2024"
- "Школьный праздник, День знаний"
- "Классные комнаты оснащены современной техникой"

## ❓ Часто задаваемые вопросы

**Q: Сколько фотографий можно добавить?**
A: Неограниченное количество! Все будут отображаться.

**Q: Что делать, если фото выглядит размыто?**
A: Загрузите исходное изображение с более высоким разрешением (1920×1440 или больше).

**Q: Фотографии обновились?**
A: Может занять несколько секунд. Нажмите F5 (обновить страницу).

**Q: Где находится страница фотогалереи?**
A: `/photo-gallery` (или с локалью `/uk/photo-gallery`, `/ru/photo-gallery`, `/en/photo-gallery`)

**Q: Могу ли я отредактировать существующую фотографию?**
A: Да! Нажмите на фотографию в Sanity и отредактируйте её. Не нужно удалять и заново добавлять.

**Q: Как удалить фотографию?**
A: В Sanity Studio рядом с фотографией нажмите иконку ✕.
