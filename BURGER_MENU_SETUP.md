# Управление Burger Menu через Sanity CMS

## 📋 Введение

Burger Menu теперь полностью управляется через Sanity CMS. Вы можете добавлять, удалять, редактировать и переупорядочивать пункты меню без изменения кода.

## 🚀 Быстрый старт

### 1. Откройте Sanity Studio

Перейдите по адресу `https://ваш-сайт.com/studio`

### 2. Найдите "Navigation Menu"

В левой боковой панели нажмите на "Navigation Menu"

### 3. Добавьте пункты меню

Нажмите кнопку "Add Menu Items" и заполните поля

## 📝 Структура пункта меню

Каждый пункт меню состоит из:

| Поле               | Тип                | Обязательное | Описание                                                                                            |
| ------------------ | ------------------ | ------------ | --------------------------------------------------------------------------------------------------- |
| **Title**          | Текст (en, ru, uk) | ✅           | Название пункта меню (может быть на разных языках)                                                  |
| **Slug**           | Строка             | ❌           | Путь в URL (например: `about`, `gallery`). Оставьте пустым, если у пункта только вложенные элементы |
| **Order**          | Число              | ❌           | Порядок сортировки (меньше = раньше в меню). По умолчанию 0                                         |
| **Sub-menu Items** | Массив             | ❌           | Вложенные пункты меню (если нужен Accordion)                                                        |

## 🎯 Примеры использования

### Пример 1: Простая ссылка

```
Title: Home (en)
Slug: home
Order: 1
Sub-menu Items: (пусто)
```

→ Отображается как обычная ссылка, ведет на `/home`

### Пример 2: Accordion с вложенными ссылками

```
Title: About (en)
Slug: about-us
Order: 2
Order: 2
Sub-menu Items:
  - Title: About Us, Slug: about, Order: 1
  - Title: Our Team, Slug: team, Order: 2
  - Title: History, Slug: history, Order: 3
```

→ Отображается как Accordion, при клике раскрывает 3 вложенных пункта

### Пример 3: Структура выпускников

```
Title: Graduates (en)
Slug: (пусто)
Order: 3
Sub-menu Items:
  - Title: 2024, Slug: graduates/2024, Order: 1
  - Title: 2025, Slug: graduates/2025, Order: 2
```

## 🌍 Поддержка языков

Каждый пункт меню поддерживает перевод на:

- 🇬🇧 **English** (en)
- 🇷🇺 **Русский** (ru)
- 🇺🇦 **Українська** (uk)

Заполните названия на нужных языках. Меню автоматически будет использовать язык, выбранный пользователем.

## ⚙️ Как это работает

1. **Сохранение в Sanity** → Вы редактируете меню в CMS
2. **Получение данных** → Frontend загружает меню через API
3. **Отображение** → Компонент BurgerMenu рендерит меню динамически
4. **Интеллектуальное отображение**:
   - Если есть вложенные пункты (Sub-menu Items) → показывает Accordion
   - Если нет вложенных пунктов → показывает простую ссылку

## ✨ Особенности

- ✅ Поддержка Accordion с вложенными ссылками
- ✅ Сортировка через поле `Order`
- ✅ Изменения применяются сразу же
- ✅ Сохранено оригинальное оформление и анимации
- ✅ Меню автоматически закрывается при клике на ссылку

## 🔄 Порядок сортировки (Order)

Все пункты меню сортируются по значению `Order`:

- `0` (или не указано) - будет в начале
- `1` - первый
- `2` - второй
- И так далее...

**Важно**: Используйте числа 10, 20, 30... если хотите легко вставлять новые пункты между существующими.

Пример:

```
Home: Order = 10
About: Order = 20
  → Sub: Order = 21
  → Sub: Order = 22
Gallery: Order = 30
Contacts: Order = 40
```

- Адаптивный дизайн

## 🚨 Важные замечания

1. **Slug должен быть уникальным** в пределах одного уровня
2. **Не используйте leading slash** в Slug (❌ `/about`, ✅ `about`)
3. **Title обязателен** для каждого пункта
4. **Вложенные пункты (children)** не могут иметь своих вложенных пунктов (максимум 2 уровня)
5. **Оставьте Slug пустым**, если пункт только для группировки вложенных ссылок

## 🐛 Решение проблем

### Меню не отображается

- ✓ Проверьте, что у вас создан документ "Navigation Menu" в Sanity
- ✓ Убедитесь, что в нем есть хотя бы один пункт меню
- ✓ Проверьте консоль браузера (F12 → Console) на ошибки

### Язык неправильный

- ✓ Убедитесь, что вы заполнили переводы на нужных языках
- ✓ Проверьте текущий язык сайта (обычно выбирается в LangSwitcher)

### Ссылки не работают

- ✓ Убедитесь, что `Slug` указан правильно
- ✓ Проверьте, что страница существует по этому маршруту

## 📞 Техническая информация

- **API Endpoint**: `/api/menu?locale=en`
- **Sanity Query**: MENU_QUERY в `sanity/lib/queries.js`
- **Компонент**: `app/components/BurgerMenu/BurgerMenu.js`
- **Схема**: `sanity/schemaTypes/menuNavigation.js`
