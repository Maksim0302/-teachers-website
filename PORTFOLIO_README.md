## 🎯 Что это?

**Portfolio** — это страница, на которой администратор может управлять коллекцией документов (PDF, Word) через Sanity CMS. Пользователи могут просматривать документы и скачивать их с веб-сайта.

### ✨ Ключевые особенности

- ✅ **Полное управление через CMS** - Без редактирования кода
- ✅ **Неограниченное количество документов** - Добавляйте сколько нужно
- ✅ **Многоязычная поддержка** - English, Ukrainian, Russian
- ✅ **Адаптивный дизайн** - Работает на всех устройствах
- ✅ **PDF Preview** - Встроенный предпросмотр для PDF
- ✅ **Download функция** - Скачивание для всех файлов
- ✅ **Автоматическая локализация** - По выбранному языку

## 🚀 Быстрый старт

### Для администратора

1. Откройте Sanity Studio
2. Создайте запись типа "Portfolio"
3. Заполните Title и Subtitle
4. Добавьте документы (PDF/DOC/DOCX)
5. Нажмите "Publish"

**[Подробнее →](./PORTFOLIO_QUICK_START.md)**

### Для разработчика

```bash
# 1. Проверить нет ошибок
npm run lint

# 2. Запустить в режиме разработки
npm run dev

# 3. Посетить страницу
http://localhost:3000/en/portfolio

# 4. Собрать для production
npm run build
```

## 🐛 Решение проблем

### Portfolio не отображается?

1. Проверьте, опубликована ли запись в Sanity
2. Убедитесь в наличии документов
3. Очистите кэш браузера

**[Больше решений →](./PORTFOLIO_ADMIN_GUIDE.md#-решение-проблем)**

### Файл не загружается?

1. Проверьте размер файла
2. Убедитесь в расширении (.pdf, .doc, .docx)
3. Проверьте соединение
