# Реализация кнопки "Повернутися на головну"

## 📋 Обзор

На все внутренние страницы проекта Next.js добавлена кнопка **"Повернутися на головну"**, которая позволяет пользователям быстро вернуться на главную страницу.

## ✅ Что было сделано

### 1. **Создан переиспользуемый компонент `BackToHome`**

- Файл: `app/components/BackToHome/BackToHome.js`
- Компонент использует:
  - `next/link` для навигации
  - `usePathname` для определения текущего маршрута
  - `useTranslations` для поддержки многоязычности
  - `react-icons/fi` для иконки стрелки влево (FiArrowLeft)

### 2. **Добавлены стили**

- Файл: `app/components/BackToHome/BackToHome.scss`
- Включает:
  - Фоновый цвет и границу
  - Плавный hover-эффект с изменением цвета
  - Анимацию стрелки (движение влево при hover)
  - Полная адаптивность для мобильных устройств (разные размеры для планшетов и смартфонов)

### 3. **Интегрирован в структуру приложения**

- Компонент добавлен в `app/[locale]/Providers.js`
- Располагается между `<Header />` и `<main>`
- Работает глобально для всех внутренних страниц

### 4. **Добавлены переводы**

- `messages/uk.json`: "Повернутися на головну"
- `messages/en.json`: "Back to Home"
- `messages/ru.json`: "Вернуться на главную"

## 🎨 Особенности дизайна

- **Иконка**: Стрелка влево (FiArrowLeft) из react-icons
- **Цвет**: Синий (#0a66c2) с темнеющей иконкой при hover
- **Адаптивность**:
  - Desktop (>1024px): стандартный размер
  - Tablet (≤1024px): уменьшенный размер
  - Mobile (≤768px): компактный размер
  - Small Mobile (≤480px): минимальный размер

## 🛠️ Технические детали

### Логика скрытия на главной странице

Компонент автоматически определяет текущий маршрут и скрывается, если пользователь находится на главной странице:

```javascript
const isHome =
  normalizedPath === '/' || (locale && normalizedPath === `/${locale}`)
if (isHome) return null
```

### Правильная навигация по локалям

Компонент корректно обрабатывает разные локали:

- `/uk/about` → кнопка ведет на `/uk`
- `/en/about` → кнопка ведет на `/en`
- `/ru/about` → кнопка ведет на `/ru`

## ✨ Результаты

Пользователи могут:

1. Видеть кнопку на всех внутренних страницах (кроме главной)
2. Быстро вернуться на главную одним нажатием
3. Автоматически вернуться на главную нужной локали
4. Видеть плавный hover-эффект на кнопке
5. Корректно использовать кнопку на мобильных устройствах
