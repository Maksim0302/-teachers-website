# 🔗 Корисні файли – Полное руководство администратора (Обновлено v2.0)

## Обзор

Страница **"Корисні посилання"** (Useful Links) теперь используется для управления коллекцией полезных **файлов (PDF и Word документы)** через **Sanity CMS** без необходимости редактирования кода. Администратор может добавлять, удалять и редактировать файлы прямо через интерфейс Sanity.

## 🔧 Инструкции для администратора Sanity

### Шаг 1: Создание записи Useful Links в Sanity

1. Откройте **Sanity Studio**
2. В левой панели найдите **"Useful Links"** или нажмите **"+ Create"**
3. Нажмите на **"Useful Links"** для создания новой записи

### Шаг 2: Заполнение основной информации

#### Поле "Section Title" (обязательное)

- **Английский**: "Useful Files"
- **Украинский**: "Корисні файли"
- **Русский**: "Полезные файлы"

#### Поле "Section Subtitle" (опциональное)

- Например: "Collection of helpful resources and documents"

### Шаг 3: Добавление полезных файлов

1. Нажмите кнопку **"+ Add item"** в секции **"Useful Files"**
2. Заполните поля для каждого файла:

#### 📄 File Title (обязательное)

Название файла (многоязычное поле):

```
EN: "Document 1"
UK: "Документ 1"
RU: "Документ 1"
```

#### 📝 File Description (опциональное)

Краткое описание (многоязычное поле):

```
EN: "Important guide"
UK: "Важний посібник"
RU: "Важный пособие"
```

#### 📎 File (обязательное)

Загрузить файл:

```
Поддерживаемые форматы: PDF, DOC, DOCX
Просто перетащите файл или нажмите для выбора
```

### Шаг 4: Опубликование

1. Убедитесь, что все обязательные поля заполнены
2. Нажмите кнопку **"Publish"** в верхнем правом углу
3. Ждите подтверждения публикации

### Функциональность кнопок

#### [Preview] (только для PDF)

- Открывает PDF в **новой вкладке**
- Позволяет просматривать документ в браузере
- Зелёная кнопка

#### [Download]

- Загружает файл на компьютер пользователя
- Работает для всех форматов (PDF, DOC, DOCX)
- Голубая кнопка

## 🔗 Добавление Useful Links в меню

### Через Sanity CMS

1. Откройте **Sanity Studio**
2. Найдите **"Menu Navigation"** в левой панели
3. Откройте существующую запись **"Menu Navigation"**
4. В секции **"Menu Items"** нажмите **"+ Add item"**

#### Заполнение элемента меню

```
Title:
  EN: "Useful Files"
  UK: "Корисні файли"
  RU: "Полезные файлы"

Slug: useful-links
Order: 5
```

5. Нажмите **"Publish"**

## 🐛 Решение проблем

### Файлы не отображаются

✅ Проверьте опубликованность записи  
✅ Убедитесь, что файлы загружены  
✅ Очистите кэш браузера  
✅ Перезагрузите страницу

### Download не работает

✅ Проверьте формат файла (PDF/DOC/DOCX)  
✅ Попробуйте в другом браузере  
✅ Проверьте консоль браузера на ошибки

### Preview PDF не работает

✅ Браузер должен поддерживать просмотр PDF  
✅ Проверьте, что это действительно PDF  
✅ Откройте файл на компьютере
