## 🚀 Быстрый старт

### 1️⃣ Администратор Sanity

```
1. Откройте http://localhost:3000/studio
2. Найдите "Photo Gallery" в левом меню
3. Нажмите "+ Create"
4. Заполните:
   - Title (заголовок на 3 языках)
   - Subtitle (подзаголовок, опционально)
   - Photos (загрузите фотографии)
5. Нажмите "Publish"
```

### 2️⃣ Откройте страницу

```
http://localhost:3000/photo-gallery
http://localhost:3000/uk/photo-gallery
http://localhost:3000/ru/photo-gallery
http://localhost:3000/en/photo-gallery
```

### 3️⃣ Готово! 🎉

Фотографии появятся автоматически в адаптивной сетке с полнофункциональным lightbox.

## 📞 Поддержка

### Часто возникающие вопросы

**Q: Как добавить больше языков?**
A: Добавить `localeString` для нового языка в Sanity, затем добавить переводы в `messages/{lang}.json`.

**Q: Как изменить количество колонок?**
A: Изменить `grid-template-columns` в `PhotoGallery.scss`.

**Q: Как добавить фильтрацию?**
A: Добавить `category` field в Sanity schema, затем добавить фильтры в компонент.

**Q: Как добавить лайки/комментарии?**
A: Использовать отдельные Sanity documents для likes и comments, связанные с фото.
