# 📖 Батьківська сторінка – Повне керівництво для адміністратора

## 🎯 Огляд

Батьківська сторінка — це місце, де адміністратор розповідає батькам про ресурси, документи та события школи. Сторінка автоматично показує:

- 📄 Документи для завантаження (розклад, звіти, вказівки)
- 📸 Фотогалерею (фото подій, вихідних днів, випускників)

---

## 🏗️ Структура Sanity Schema

### Основні поля

| Поле                 | Тип   | Опис                               |
| -------------------- | ----- | ---------------------------------- |
| **Section Title**    | Text  | Заголовок сторінки (локалізований) |
| **Section Subtitle** | Text  | Підзаголовок (локалізований)       |
| **Documents**        | Array | Масив документів                   |
| **Photo Gallery**    | Array | Масив фотографій                   |

---

## 📝 Детальне керівництво

### 1️⃣ Налаштування Заголовків

#### Section Title

```
En: Parents Page
Uk: Батьківська сторінка
Ru: Страница для родителей
```

**Як заповнити:**

- Натисніть на поле **Title**
- Виберіть мову з вкладок (EN, UK, RU)
- Введіть текст
- Натисніть Publish

#### Section Subtitle

```
En: Resources and documents for parents
Uk: Ресурси та документи для батьків
Ru: Ресурсы и документы для родителей
```

---

### 2️⃣ Додавання Документів

#### Етап 1: Откройте розділ Documents

1. Прокрутіть вниз до **"Documents for Parents"**
2. Натисніть **"Add item"** (зелена кнопка)

#### Етап 2: Заповніть базові дані

| Поле               | Обов'язкове | Приклад                                   |
| ------------------ | ----------- | ----------------------------------------- |
| **Document Title** | ✅ Так      | Розклад уроків 2024-2025                  |
| **Description**    | ❌ Ні       | Офіційний розклад на новий навчальний рік |
| **File**           | ✅ Так      | schedule.pdf                              |

**Локалізація Title:**

- Натисніть на **Document Title**
- Додайте текст для кожної мови

#### Етап 3: Завантажте файл

1. Натисніть на **File** поле
2. Виберіть файл (PDF, DOC або DOCX)
3. Зачекайте завантаження

#### Етап 4: Збережіть

- Натисніть **Publish** у верхньому правому куті

---

### 3️⃣ Додавання Фотографій

#### Етап 1: Откройте розділ Photo Gallery

1. Прокрутіть вниз до **"Photo Gallery"**
2. Натисніть **"Add item"**

#### Етап 2: Додайте зображення

| Поле         | Опис                                   |
| ------------ | -------------------------------------- |
| **Photo**    | Завантажте JPG, PNG або WEBP           |
| **Alt Text** | Альтернативний текст (важливо для SEO) |

**Приклад Alt Text:**

```
En: Student presentation at the parent meeting
Uk: Презентація учня на батьківському зібранні
Ru: Презентация ученика на родительском собрании
```

#### Етап 3: Збережіть

- Натисніть **Publish**

## 🔄 Управління Контентом

### Редагування існуючого документу

1. **Откройте запис** батьківської сторінки
2. **Знайдіть документ** у списку "Documents for Parents"
3. **Натисніть на документ** щоб розкрити його
4. **Змініть необхідні поля**
5. **Натисніть Publish**

### Видалення документу

1. **Знайдіть документ** у списку
2. **Натисніть на крестик (✕)** праворуч
3. **Натисніть Publish**

### Зміна порядку документів

1. **Утримуйте** документ за **≡ (drag handle)**
2. **Перетягніть** на потрібне місце
3. **Натисніть Publish**

## 🐛 Вирішення проблем

### Проблема: Документ не завантажується

**Рішення:**

1. Перевірте формат файлу (PDF/DOC/DOCX)
2. Перевірте розмір файлу (менше 100 MB)
3. Спробуйте завантажити заново

### Проблема: Фотографія розмита

**Рішення:**

1. Виберіть зображення вищої якості
2. Рекомендуємий розмір: 1200x800px мінімум

### Проблема: Текст не локалізується

**Рішення:**

1. Перевірте, що поле має вкладки EN, UK, RU
2. Заповніть всі вкладки текстом
