## ✨ Ключевые особенности v2.0

### 📄 PDF Preview

- Кнопка "Preview" появляется только для PDF файлов
- Открывает PDF в новой вкладке
- Встроенный просмотр браузером

### 📥 Download для всех файлов

- Работает для PDF, DOC, DOCX
- Кнопка "Download" всегда видна
- Native browser download API

### 🎯 Автоматическое определение типа

- Анализирует расширение файла
- Показывает правильную иконку (📄 PDF, 📘 Word)
- Определяет доступность Preview

### 🌐 Многоязычная поддержка

- Все языки (EN, UK, RU) поддерживаются
- Локализованные названия кнопок
- Переводы для пустого состояния

### 📱 Адаптивный дизайн

- Мобильные устройства: одна колонка
- Планшеты: оптимальное расположение
- Десктоп: полный размер

## 🔄 Миграция с v1.0 на v2.0

### Что нужно сделать

1. **Обновить Sanity документ**

   ```
   Удалить старое поле "links" с URL
   Добавить новое поле "files" с загруженными файлами
   ```

2. **Опубликовать изменения**

   ```
   Нажать "Publish" в Sanity
   ```

3. **Очистить кэш браузера**

   ```
   Ctrl+Shift+Delete (Windows/Linux)
   Cmd+Shift+Delete (Mac)
   ```

4. **Проверить на всех языках**
   ```
   /en/useful-links
   /uk/useful-links
   /ru/useful-links
   ```

### Обратная совместимость

❌ **NOT COMPATIBLE** с v1.0  
🔄 Нужна миграция данных если были URL ссылки
