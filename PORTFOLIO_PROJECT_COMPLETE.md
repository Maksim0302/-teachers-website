## 🎯 Как использовать

### Для администратора (30 мин)

1. **Прочитайте** `PORTFOLIO_QUICK_START.md` (5 мин)
2. **Создайте** Portfolio в Sanity (10 мин)
3. **Добавьте** документы (зависит от количества)
4. **Опубликуйте** (1 мин)
5. **Проверьте** на сайте (2 мин)

### Для разработчика (1-2 часа)

1. **Прочитайте** `PORTFOLIO_IMPLEMENTATION.md` (30 мин)
2. **Запустите** локально `npm run dev` (5 мин)
3. **Протестируйте** используя `PORTFOLIO_TESTING_GUIDE.md` (30 мин)
4. **Разверните** на production (15 мин)
