# Photo Gallery - Обновление стилизации

## 🎨 Что изменилось

Стилизация страницы PhotoGallery теперь **полностью идентична** странице GraduatesPageContent.scss

### Ключевые изменения

#### 1. **Компонент PhotoGallery.js**

- ✅ Упрощена логика модального окна (вместо сложного lightbox с навигацией)
- ✅ Используется простой `selectedPhoto` state вместо `selectedPhotoIndex`
- ✅ Убран компонент `PhotoGalleryLightbox`
- ✅ Модальное окно встроено прямо в основной компонент
- ✅ Добавлена поддержка Esc для закрытия модального окна
- ✅ Используются простые `<img>` элементы вместо Next.js Image компонента (как в Graduates)

#### 2. **Сетка фотографий**

**До:**

```scss
grid-template-columns: repeat(4, 1fr);      // Всегда 4 колонки
@media (max-width: 1024px) { 3fr }          // 3 колонки
@media (max-width: 768px) { 2fr }           // 2 колонки
@media (max-width: 480px) { 1fr }           // 1 колонка
```

**После (как в Graduates):**

```scss
grid-template-columns: repeat(auto-fit, minmax(320px, 1fr));
// Автоматически адаптируется в зависимости от ширины экрана
// Минимум 320px на элемент, максимум 1fr
```

#### 3. **Соотношение сторон (Aspect Ratio)**

**До:**

```scss
aspect-ratio: 1; // Квадратные фотографии
```

**После:**

```scss
aspect-ratio: 4 / 3; // Прямоугольные (как стандартный фото)
```

#### 4. **Модальное окно**

**До:**

```
- Сложный lightbox с:
  - Стрелками навигации ← →
  - Счетчиком фотографий (1/10)
  - Подписями внизу
  - Закрытием по Esc, клику на фон и кнопке X
```

**После (как в Graduates):**

```
- Простое модальное окно с:
  - Только кнопкой закрытия (✕) в углу
  - Закрытием по Esc, клику на фон и кнопке X
  - Плавной анимацией fadeIn/zoomIn
```

#### 5. **Стили и анимации**

**Добавлены:**

- ✅ `fadeIn` анимация для модального окна (0.25s)
- ✅ `zoomIn` анимация для изображения в модалке (0.25s)
- ✅ `translateY(-4px)` при hover на фотографию (вместо scale)
- ✅ `scale(1.03)` при hover на изображение (вместо overlay с лупой)

**Удалены:**

- ❌ Overlay с иконкой 🔍
- ❌ Подписи под фотографиями в сетке
- ❌ Loading состояния с opacity переходами

#### 6. **Hover эффекты на мобилке**

```scss
@media (max-width: 768px) {
  .photo-gallery__photo-item:hover {
    transform: none; // Нет transform на мобилке
  }

  .photo-gallery__photo:hover {
    transform: none; // Нет scale на мобилке
  }
}
```

## 📱 Адаптивность

### Размеры сетки:

- **ПК (>1024px)**: Автоматически 3-4 фотографии в ряд (по 320px каждая)
- **Планшет (768-1024px)**: 2-3 фотографии в ряд (по 240px минимум)
- **Мобила (<768px)**: 1-2 фотографии в ряд (по 240px минимум)
- **Смартфон (<480px)**: 1 фотография в ряд (полная ширина)

### Зазоры (gap):

- **ПК/Планшет**: 24px
- **Мобила**: 16px
- **Смартфон**: 12px (по умолчанию через media query)

## 🎯 Функциональность

### Открытие фотографии:

1. **Клик на фотографию** → открывает в полноэкранном режиме
2. **Клик на ✕** → закрывает модальное окно
3. **Клик на черный фон** → закрывает модальное окно
4. **Нажатие Esc** → закрывает модальное окно

### Hover эффекты:

- **На ПК**: Фотография поднимается на 4px вверх, изображение увеличивается на 3%
- **На мобилке**: Без hover эффектов (экономия батареи/производительности)
