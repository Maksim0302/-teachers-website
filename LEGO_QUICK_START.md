# 🚀 LEGO - конструювання – Швидкий старт

## ⏱️ Займе 5 хвилин!

### Крок 1️⃣ Відкрийте Sanity Studio

```bash
npm run dev
# або
sanity dev
```

Перейдіть на http://localhost:3333 (або Sanity Studio)

---

### Крок 2️⃣ Створіть нову запис

1. **У ліворуч меню** знайдіть **"LEGO - конструювання"**
2. **Натисніть "Create"** або **"+"**
3. Заповніть основні поля:

| Поле                 | Приклад                                      |
| -------------------- | -------------------------------------------- |
| **Section Title**    | LEGO - конструювання                         |
| **Section Subtitle** | Матеріали для конструювання та навчання LEGO |

---

### Крок 3️⃣ Додайте Документи та Фото

#### 📄 Додавання Документів

1. Прокрутіть до **"Documents"**
2. **Натисніть "Add item"**
3. Заповніть:
   - **Document Title**: Назва документа (наприклад, "Інструкція зі збирання")
   - **Description** (опціонально): Короткий опис
   - **File**: Завантажте PDF, DOC або DOCX

4. **Натисніть Publish**

#### 📸 Додавання Фотографій

1. Прокрутіть до **"Photo Gallery"**
2. **Натисніть "Add item"**
3. Заповніть:
   - **Photo**: Завантажте зображення
   - **Alt Text**: Альтернативний текст (для SEO)

4. **Натисніть Publish**
