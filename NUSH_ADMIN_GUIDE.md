# 📚 НУШ (Нова українська школа) – Полное руководство администратора

## Обзор

Страница **НУШ** позволяет полностью управлять коллекцией документов (PDF и Word) для **Nova Ukrainska Shkola (Новая украинская школа)** через **Sanity CMS** без необходимости редактирования кода. Администратор может добавлять, удалять и редактировать документы прямо через интерфейс Sanity.

## 🔧 Инструкции для администратора Sanity

### Шаг 1: Создание записи НУШ в Sanity

1. Откройте **Sanity Studio**
2. В левой панели найдите **"NUŠ"** или нажмите **"+ Create"**
3. Нажмите на **"NUŠ"** для создания новой записи

### Шаг 2: Заполнение основной информации

#### Поле "Section Title" (обязательное)

- **Английский**: "NUŠ Documents"
- **Украинский**: "НУШ Документи"
- **Русский**: "НУШ Документы"

#### Поле "Section Subtitle" (опциональное)

- **Английский**: "Nova Ukrainska Shkola resources"
- **Украинский**: "Матеріали Нової Української Школи"
- **Русский**: "Материалы Новой Украинской Школы"

### Шаг 3: Добавление документов

1. Нажмите кнопку **"+ Add item"** в секции **"NUŠ Documents"**
2. Заполните поля для каждого документа:

#### 📄 Document Title (обязательное)

Название документа (многоязычное поле):

```
EN: "Teaching Recommendations"
UK: "Методичні рекомендації"
RU: "Методические рекомендации"
```

#### 📝 Document Description (опциональное)

Краткое описание (многоязычное поле):

```
EN: "Recommendations for teachers"
UK: "Рекомендації для вчителів"
RU: "Рекомендации для учителей"
```

#### 📎 Document File (обязательное)

Загрузить файл:

```
Поддерживаемые форматы: PDF, DOC, DOCX
Просто перетащите файл или нажмите для выбора
```

### Шаг 4: Опубликование

1. Убедитесь, что все обязательные поля заполнены
2. Нажмите кнопку **"Publish"** в верхнем правом углу
3. Ждите подтверждения публикации

### Функциональность кнопок

#### [Preview] (только для PDF)

- Открывает PDF в **новой вкладке**

- Зелёная кнопка

- Работает для всех форматов (PDF, DOC, DOCX)
- Синяя кнопка

## 🔗 Добавление НУШ в меню

### Через Sanity CMS

1. Откройте **Sanity Studio**
2. Найдите **"Menu Navigation"** в левой панели
3. Откройте существующую запись **"Menu Navigation"**
4. В секции **"Menu Items"** нажмите **"+ Add item"**

#### Заполнение элемента меню

```
Title:
  EN: "NUŠ"
  UK: "НУШ"
  RU: "НУШ"

Slug: nush
Order: [ваша позиция в меню]
```

5. Нажмите **"Publish"**

## 🐛 Решение проблем

### Документы не отображаются

✅ Проверьте опубликованность записи  
✅ Убедитесь, что документы загружены  
✅ Очистите кэш браузера  
✅ Перезагрузите страницу

### Download не работает

✅ Проверьте формат файла (PDF/DOC/DOCX)  
✅ Попробуйте в другом браузере  
✅ Проверьте консоль браузера на ошибки (F12)

### Preview PDF не работает

✅ Браузер должен поддерживать просмотр PDF  
✅ Проверьте, что это действительно PDF  
✅ Откройте файл на компьютере

## 📝 Технические детали

### 1. Schema (Sanity)

```javascript
// sanity/schemaTypes/nush.js
- Type: document
- Fields: title, subtitle, documents (array)
- Document fields: title, description, file
- File types: PDF, DOC, DOCX
```

### 2. GROQ Query

```groq
*[_type == "nush"][0]{
  title,
  subtitle,
  documents[]{
    title,
    description,
    file{
      asset->{
        originalFilename,
        url
      }
    }
  }
}
```

### 3. Frontend Routes

```
/nush → redirect to /{locale}/nush
/{locale}/nush → Page with NUŠ content
```
