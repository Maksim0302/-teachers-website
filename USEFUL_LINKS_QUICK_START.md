# 🔗 Корисні посилання – Quick Start

> **Хотите быстро начать работу с Useful Files? Это руководство для вас!**

---

## 🚀 3 простых шага

### Шаг 1️⃣: Создать Useful Files в Sanity

1. Откройте **Sanity Studio**
2. Нажмите **"+ Create"** (или найдите **Useful Links** в левом меню)
3. Создайте новую запись

### Шаг 2️⃣: Заполнить информацию

```
Title (Название):
  EN: "Useful Files"
  UK: "Корисні файли"
  RU: "Полезные файлы"

Subtitle (Подзаголовок) - опционально:
  EN: "Collection of helpful resources"
```

### Шаг 3️⃣: Добавить файлы

1. Нажмите **"+ Add item"** в **"Useful Files"**
2. Заполните для каждого файла:
   - **File Title**: Название (например, "Документ 1")
   - **File Description**: Описание (опционально)
   - **File**: PDF или Word файл (обязательно)
3. Нажмите **"Publish"**

---

## 📍 Где будет видна страница

После добавления в меню:

```
http://example.com/useful-links
→ Перейдет на http://example.com/en/useful-links
→ или http://example.com/uk/useful-links
→ или http://example.com/ru/useful-links
```

## ❓ Часто задаваемые вопросы

**Q: Какие форматы поддерживаются?**  
A: PDF, DOC, DOCX

**Q: Сколько файлов можно добавить?**  
A: Неограниченно! Добавляйте сколько нужно

**Q: Как удалить файл?**  
A: Удалите элемент из массива и нажмите "Publish"

**Q: Можно ли предпросмотреть PDF?**  
A: Да, для PDF файлов доступна кнопка "Preview" (откроется в новой вкладке)
